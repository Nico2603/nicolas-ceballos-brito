# 🚀 Portafolio Personal de Nicolás Ceballos Brito

## 👨‍💻 Sobre mí

Soy estudiante de **décimo semestre** de **Ingeniería de Sistemas y Telecomunicaciones** en la Universidad Católica de Pereira. Con amplia experiencia en desarrollo de software, análisis de datos e inteligencia artificial, he participado en diversos proyectos que abarcan desde machine learning para mantenimiento predictivo hasta el desarrollo de aplicaciones web interactivas.

Mi participación como **representante estudiantil** y en **semilleros de investigación** me ha permitido obtener certificaciones en SQL, Power BI, UX/UI y metodologías ágiles, fortaleciendo mi perfil técnico y de liderazgo.

Este repositorio contiene el código fuente de mi página web personal, que sirve como portafolio profesional donde muestro mis proyectos más destacados, experiencia y habilidades técnicas.

## ✨ Características Principales

### 🎨 Diseño y UX
- **Diseño Responsivo**: Optimizado para todos los dispositivos (móvil, tablet, desktop)
- **Interfaz Moderna**: Uso de glassmorphism, gradientes y microinteracciones
- **Navegación Intuitiva**: Menú principal con dropdowns organizados por categorías
- **Animaciones Fluidas**: Transiciones suaves y efectos parallax sutiles
- **Modo Oscuro**: Esquema de colores optimizado para la experiencia visual

### 🚀 Funcionalidades Avanzadas
- **Carrusel Interactivo**: Muestra 5 labores destacadas con navegación manual y automática
- **Integración GitHub API**: Carga dinámica de repositorios con filtros y estadísticas
- **Single Page Application (SPA)**: Navegación fluida entre secciones
- **Explorador de Repositorios**: Página dedicada con filtros avanzados por tecnología
- **Optimización SEO**: Metadatos estructurados y schema markup
- **Performance**: Lazy loading de imágenes y optimización de recursos

### 📱 Experiencia Móvil
- **Menú Hamburguesa**: Navegación optimizada para dispositivos táctiles
- **Dropdowns Móviles**: Submenús expandibles en dispositivos pequeños
- **Touch Gestures**: Navegación por swipe en el carrusel
- **Optimización de Rendimiento**: Deshabilitación de efectos pesados en móviles

## 🚀 Cómo Utilizarlo

1. **Clona el repositorio:**
   ```bash
   git clone https://github.com/Nico2603/PersonalPage.git
   cd PersonalPage
   ```

2. **Abre el archivo `index.html` en tu navegador** para ver el sitio localmente.

3. **Explora las diferentes páginas:**
   - `index.html` - Página principal con carrusel y proyectos
   - `pages/aboutme.html` - Información detallada sobre habilidades
   - `pages/repositories.html` - Explorador completo de repositorios

## ⚙️ Personalización

### Cambiar Usuario de GitHub
Para mostrar tus propios repositorios, edita la constante en `assets/js/app.js`:
```javascript
const GITHUB_USERNAME = 'TuUsuarioDeGitHub';
```

### Modificar Proyectos Destacados
Edita la sección `.projects-grid` en `index.html` para mostrar tus propios proyectos con sus respectivas tecnologías y estadísticas.

### Actualizar Carrusel de Labores
1. Reemplaza las imágenes en `assets/images/` (p1.jpg a p5.jpg)
2. Modifica el contenido del carrusel en la sección `#labores`
3. Ajusta las descripciones y enlaces según tus experiencias

### Personalizar Información
- **Página principal:** Edita las secciones en `index.html`
- **Sobre mí:** Modifica `pages/aboutme.html` con tu información personal y profesional
- **Habilidades:** Actualiza las tecnologías y certificaciones en la página aboutme
